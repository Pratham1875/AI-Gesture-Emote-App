# AI Gesture & Face Controlled Emote App

A real-time computer vision application built using Python, OpenCV, and MediaPipe that detects hand gestures and facial expressions to trigger multimedia reactions.

## Features
- Multi-hand gesture recognition (Namaste, Peace, Double Index, Thumbs Up)
- Face gesture detection (Tongue Out)
- Real-time MediaPipe landmark tracking
- Modular architecture (Hand + Face trackers)
- Cooldown-based multimedia triggering
- OBS Virtual Camera compatible

## Tech Stack
- Python
- OpenCV
- MediaPipe
- VLC / Video playback integration
